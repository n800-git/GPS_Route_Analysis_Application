# GPS Route Analysis Assignment

## Project Report

---

## Objective

Develop a modular Python application capable of processing GPS datasets and generating useful analytical outputs.

---

## Input

CSV GPS datasets.

Typical fields include:

- Timestamp
- Latitude
- Longitude
- Altitude
- Speed
- Heading

---

## Processing Pipeline

Dataset Discovery

↓

CSV Validation

↓

Data Cleaning

↓

Route Analysis

↓

Map Generation

↓

GeoJSON Export

↓

HTML Report

↓

Logging

---

## Route Statistics

The application computes:

- Total Distance
- Mission Duration
- Maximum Altitude
- Minimum Altitude
- Average Altitude
- Average Speed
- Maximum Speed
- Average Heading

---

## Outputs

- Interactive HTML Map
- GeoJSON Route
- HTML Report
- Processing Log

---

The application contains:

- Unit Tests
- Integration Test

All tests pass successfully.
